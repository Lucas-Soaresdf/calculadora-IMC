# Calculadora IMC

Projeto desenvolvido com HTML, CSS e JavaScript com o objetivo de calcular o Índice de Massa Corporal (IMC) de forma simples, dinâmica e responsiva.

Durante o desenvolvimento pratiquei manipulação do DOM, validação de dados, eventos e lógica de programação, além de trabalhar na criação de uma interface amigável para diferentes tamanhos de tela.

## Funcionalidades

* Cálculo automático do IMC
* Validação de dados inseridos
* Exibição dinâmica do resultado
* Mensagens de erro para dados inválidos
* Interface responsiva

## Linguagens

* HTML5
* CSS3
* JavaScript
